####2048 GAME

####개선 내용
>1. Design – Background color 변경
>2. Highest score - 최고점과 현재 점수 비교
>3. Button operation - 버튼 기능의 세부적인 추가(Quit, Restart, Yes, No)로 사용자의 편의성 향상
>4. Show Button information - 버튼 기능을 게임 시작과 동시에 사용자에게 보여줌으로써, 게임 작동 방법을 쉽게 익힐 수 있음
>5. Restart game operation after termination – 게임이 완전히 끝난 후에도, 다시 한 번 더 게임을 할지에 대한 여부를 물어봄으로써 더 적극적인 게임 참여 유도 가능
 
####개선 과정에서 힘들었던 점
>1. 기존 코드를 변형하는 형태이다 보니, 전체적인 코드를 이해하는 데 걸리는 시간의 소모
>2. 2048 게임 특성상 더 업그레이드할 점이 마땅치 않음
>3 . 리눅스 터미널 게임이라  UI 를 다루는데 익숙하지 않음  – R GB 같은 요소를 모두 분석해서 수정해야하는 번거로움
